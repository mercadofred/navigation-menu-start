The Markdown Cheat Sheet, part of the Markdown Guide is a free and open-source reference guide that explains how to use Markdown, the simple and easy-to-use markup language you can use to format virtually any document. Markdown can be used instead of HTML to create websites.

To be clear, we won't be using Markdown as such to create web content, but instead will use the Markdown Cheat Sheet as a reference to the HTML commands one can use to build webpages.
